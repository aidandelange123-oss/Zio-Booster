📚 API Documentation

Installation

<!-- Include the script in your HTML --> <script src="video-browser-api.js"></script>

Basic Usage

// Create a new player instance const player = new ZioBoosterPlayer({ container: '#video-container', videoUrl: 'https://example.com/video.mp4', watermarkText: 'Zio Booster', autoplay: true }); // Or use the API helper const player = ZioBoosterAPI.playVideo('#container', 'video.mp4', { watermarkText: 'My Brand' });

Configuration Options

  • container - DOM element or selector for the player
  • videoUrl - URL of the video file
  • watermarkText - Text to display as watermark (default: "Zio Booster")
  • watermarkPosition - Position: top-left, top-right, bottom-left, bottom-right, center
  • watermarkOpacity - Opacity level from 0 to 1 (default: 0.7)
  • watermarkColor - Color in hex format (default: #ffffff)
  • watermarkFontSize - Font size with units (default: 24px)
  • autoplay - Auto-play video on load (default: false)
  • showControls - Show/hide custom controls (default: true)

Available Methods

  • loadVideo(url) - Load a new video
  • play() - Play the video
  • pause() - Pause the video
  • togglePlay() - Toggle play/pause
  • seekTo(position) - Seek to position (0-1)
  • seekRelative(seconds) - Seek forward/backward
  • setVolume(level) - Set volume (0-1)
  • toggleMute() - Toggle mute
  • toggleFullscreen() - Toggle fullscreen
  • setWatermark(text) - Update watermark text
  • setWatermarkOptions(options) - Update watermark settings
  • destroy() - Remove player and cleanup

Event Callbacks

  • onReady(player) - Called when player is initialized
  • onPlay(player) - Called when video starts playing
  • onPause(player) - Called when video is paused
  • onEnded(player) - Called when video ends
  • onError(error, player) - Called on error

Keyboard Shortcuts

  • Space / K - Play/Pause
  • F - Toggle Fullscreen
  • M - Mute/Unmute
  • ← - Rewind 5 seconds
  • → - Forward 5 seconds
